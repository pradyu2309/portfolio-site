CloudNote

CloudNote is a secure, full-stack cloud-based note-taking web application. It allows users to register, login, and manage personal notes with full CRUD functionality. Notes are private per user and securely stored using JWT and PostgreSQL. Includes Swagger UI and a beautiful single-page dashboard interface.

  • Spring Boot backend with JWT Authentication
  • PostgreSQL for user & note storage
  • Frontend with HTML/CSS/JS and responsive dashboard
  • Swagger UI + Postman for API testing
  • User-specific notes and profile with dynamic views

Tech Stack: Java, Spring Boot, PostgreSQL, HTML, CSS, JavaScript
